How to enable GPS (if location is blocked)
The prompt names the browser/website, not this app, because installed web apps use the browser's location permission. If you tapped “Don’t Allow,” re-enable it:
iPhone — Safari: Settings → Apps → Safari → Location → Allow. Also check Settings → Privacy & Security → Location Services is On. Then reopen the app and tap GPS Locate.
iPhone — Chrome (or Edge/Firefox): Settings → Apps → Chrome → Location → Allow (substitute your browser's name). Location Services must also be On as above.
Android — Chrome: tap the ⋮ menu → Settings → Site settings → Location → make sure it's allowed, and remove this site from “Blocked” if listed. Also Android Settings → Location → On, and Settings → Apps → Chrome → Permissions → Location → Allow.
Android — installed app (Add to Home screen): long-press the icon → App info → Permissions → Location → Allow.
Tip: the very first GPS prompt appears when you tap GPS Locate — do it once while you still have signal to get the permission out of the way and speed up later cold fixes.

DATA & ACCURACY — READ FIRST
This app is a beta, for experimental use only. All data is unverified and should not be relied on. It is a planning toy, not a navigation, safety, or decision-making tool. Always carry and use a dedicated navigation app (FarOut, Gaia, CalTopo) and paper backup.
Sources. Trail centerline: Pacific Crest Trail Association (PCTA) published route data. Elevation: USGS / NASA SRTM public-domain terrain models, sampled via GPS Visualizer. Waypoints and mile markers: compiled by hand and approximate.
Known caveats. Mile markers can differ from FarOut/Halfmile by a mile or more (trail reroutes; calibrated only at ~36 trailheads). Elevation is sampled every 0.1 mi and smoothed — summits read slightly low and daily climb totals are estimates. Water sources, resupply status, trail closures, snow, and fire conditions are NOT tracked here and change constantly. Trail access rules change (e.g., entry into Canada is currently not permitted). Verify everything against current official sources before acting on it.
